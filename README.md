# WeatherX — Modern Weather App

Beautiful, responsive aur bilkul clean weather app built with React + OpenWeatherMap API

**Live Demo** → (yahan apna Vercel/Netlify link daal dena)

## Features

- Real-time current weather (temperature, feels like, humidity, wind)
- Mausam ke hisaab se background automatically change hota hai
- Hourly forecast (next 12 hours)
- 5-Day forecast with high/low
- Accurate AQI (0-500 scale) with color indicator
- °C / °F toggle
- Search any city in the world
- Current location se weather (GPS)
- 100% mobile responsive
- Glassmorphism + modern UI (zero external library)

## Tech Stack

- React 18 (Hooks)
- OpenWeatherMap API
- Vanilla CSS (no Tailwind, no Bootstrap)
